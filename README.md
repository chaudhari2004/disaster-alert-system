# 🚨 DisAlert — Smart Disaster Alert & Emergency Response System

AWS Cloud • Python • Flask • Serverless • Emergency Automation

---

## ✨ About This Project

DisAlert is a cloud-based disaster alert and emergency response platform developed using Python, Flask, and AWS Services.

The system helps users receive instant disaster alerts and send emergency SOS requests during emergencies such as floods, earthquakes, fires, and other disasters.

---

## 🚀 Key Features

- 🚨 Real-Time Disaster Alerts
- 🆘 SOS Emergency Request System
- ☁️ AWS Cloud Integration
- 📢 AWS SNS Notifications
- 🗄️ DynamoDB Database
- 📦 AWS S3 Storage
- ⚡ AWS Lambda Automation
- 📊 Monitoring Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/chaudhari2004/disaster-alert-system.git
```

### Move to Project Folder

```bash
cd disaster-alert-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
