✨ About This Project

DisAlert is a cloud-based disaster alert and emergency response platform developed using Python, Flask, and AWS Services.

The system helps users receive instant disaster alerts and send emergency SOS requests during critical situations such as floods, earthquakes, fires, and other emergencies.

This project demonstrates practical implementation of:

✅ AWS Cloud Services
✅ Real-Time Alert Systems
✅ Serverless Architecture
✅ Python Web Development
✅ Emergency Communication Automation
✅ Cloud-Based Monitoring
