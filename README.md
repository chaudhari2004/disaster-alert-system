🚨 DisAlert — Disaster Alert & Emergency Response System

DisAlert is a cloud-based disaster management and emergency response system developed using Python, Flask, and AWS Services. The platform helps users receive real-time disaster alerts and send SOS emergency requests during critical situations.

📌 Features
🚨 Real-time disaster alerts
🆘 SOS emergency request system
☁️ AWS cloud integration
📢 AWS SNS notification service
🗄️ DynamoDB database integration
📦 AWS S3 storage support
⚡ AWS Lambda serverless processing
📊 Simple dashboard interface
🛠️ Technologies Used
Python
Flask
HTML/CSS
AWS SNS
AWS DynamoDB
AWS S3
AWS Lambda
