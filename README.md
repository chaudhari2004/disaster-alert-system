# ✨ About The Project

DisAlert is a cloud-based disaster alert and emergency response platform developed using **Python, Flask, and AWS Services**.

The system helps users receive instant disaster alerts and send emergency SOS requests during critical situations such as:

- 🌊 Floods
- 🌍 Earthquakes
- 🔥 Fires
- 🌪️ Natural Disasters
- 🚨 Emergency Situations

The project demonstrates practical implementation of:

✅ AWS Cloud Services  
✅ Real-Time Alert Systems  
✅ Serverless Architecture  
✅ Python Web Development  
✅ Emergency Communication Automation  
✅ Cloud Monitoring

# ⚙️ Installation & Setup

## 📥 Clone Repository

```bash
git clone https://github.com/chaudhari2004/disaster-alert-system.git
```

## 📂 Move To Project Folder

```bash
cd disaster-alert-system
```

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run Application

```bash
python app.py
```
